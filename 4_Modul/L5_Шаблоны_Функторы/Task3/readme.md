# Задача 3. Счётчик чисел

### Описание
В этом задании вам нужно реализовать функтор, который подсчитывает сумму и количество чисел, делящихся на 3. 

Должны быть реализованы 2 функции:
* get_sum()
* get_count()

---

### Пример правильной работы программы
```
[IN]: 4 1 3 6 25 54
[OUT]: get_sum() = 63
[OUT]: get_count() = 3
```
